ICUIMC2010にて6件の発表を行いました

2010-01-18

ICUIMC2010(2010.1.14,15@Sungkyunkwan University, Suwon, Korea)にて、
以下の6件の発表を行いました。
外は氷点下で雪も厚く積もる極寒でしたが、会場では朝早くから夕方まで熱い議論が交わされました。
各自の発表も、完成度が高いものになっており、英語発表の貴重な経験を得ました。

・Evaluating Credibility of Web Information
Katsumi Tanaka, Hiroaki Ohshima, Adam Jatowt, Satoshi nakamura, Masatoshi Yoshikawa,
Yasuhito Asano, Qiang Ma, Kazutoshi Sumiya

・ChronoSeeker: Search Engine for Future and Past Events
Hideki Kawai, Adam Jatowt, Katsumi Tanaka, Kazuo Kunieda, Keiji Yamada

・Intent Feature Discovery using Q&A Corpus and Web Data
Soungwoong Yoon, Adam Jatowt, Katsumi Tanaka

・Query by Example for Geographic Entity Search with Implicit Negative Feedback
Makoto P. Kato, Hiroaki Ohshima, Satoshi Oyama, Katsumi Tanaka

・Adaptive Ranking of Search Results by Considering User’s Comprehension
Makoto Nakatani, Adam Jatowt, Katsumi Tanaka

・Discovering Intermediate Entities from Two Examples by Using Web Search Engine Indices
Naoto Asahi, Takehiro Yamamoto, Satoshi Nakamura, Katsumi Tanaka
