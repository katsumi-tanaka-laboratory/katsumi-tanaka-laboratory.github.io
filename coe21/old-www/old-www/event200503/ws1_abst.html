異メディア・アーカイブの 横断的検索・統合ソフトウエアの開発
Software Development of Cross-Retrieval & Integration for Heterogeneous-Media Digital Archive
田中克己（京都大学情報学研究科）
Katsumi Tanaka (Graduate School of Informatics, Kyoto University)
2004年度から5年間の予定でスタートした文部科学省科学技術振興費プロジェクト「異メディア・アーカイブの横断的検索・統合ソフトウェア開発」（代表：京都大学田中克己）について、その概要、および、京都大学が主に担当するクロスメディア検索エンジンについて、ご紹介します。本プロジェクトは、さまざまなメディア形態でアーカイブされているコンテンツを横断的に検索・収集するソフトウエアを開発し、そのようなデジタルアーカイブ・コンテンツを教育分野に利活用することをねらいとしています。具体的には、美術館・博物館などの文化遺産の画像アーカイブ、関連する放送映像アーカイブ、及びWeb情報などを、目的に応じて横断的に検索・収集しeラーニングへの活用が可能な、クロスメディア・サーチエンジンを開発します。京都大学では、全体の総括を行うと共に、異メディアコンテンツのクロスメディア検索エンジンを開発します。また、意味的・感性的連想サーチエンジンを慶應大学が、セマンティックWebによる統合エンジンを富士通が、文化財コンテンツ統合のためのメタデータ構造の確立を国立情報学研究所が、それぞれ担当します。京都大学が開発予定のクロスメディア検索エンジンは、利用者の入力したキーワード質問を分解・修正して、既存の各メディアに対応したサーチエンジンに質問を分配・実行し、収集・検索されたコンテンツの意味内容を自動分析して統合するメタサーチエンジンで、この概要についても紹介をします。

The 5-year “Intelligent Information Resource Projects” by the Ministry of Education, Culture, Sports, Science and Technology started from the fiscal year of 2004. In this talk, we will describe the overview of one of those projects, called “Software Development for Cross-Retrieval & Integration over Heterogeneous-Media Digital Archive” lead by Professor Katsumi Tanaka, KyotoUniversity. Also, we will introduce the “cross-media search engine” that has been mainly developed by KyotoUniversity for this project. The aim of our project is to develop the software to collect and retrieve distributed digital archive of heterogeneous media types, and to promote the usage of digital archive for education and e-learning. The target content of our project is not only the cultural heritage digital archive stored at museums and art libraries, but also related broadcasting video contents and Web contents. Under this project, KeioUniversity is responsible for developing semantic & impression (“Kansei”) based search engines, Fujitsu is for developing an information integration engine based on semantic Web technology, and the National Institute of Informatics (NII) is for establishing a metadata framework for integrating cultural heritage digital archive. We will describe the “cross-media search engine” under development at KyotoUniversity. It is a new type of meta-search engine, which automatically relaxes, converts, and modifies a user-given query into appropriate queries for each search engine of different media type, and integrates the retrieval results by analyzing their contents.
 
感性によるクロスメディア型コンテンツサーチ技術と
知的資産活用のためのヒューマンインターフェース技術
Cross-Media Contents Search Technology for "Kansei" Information
and Human Interface for Intellectual Property
清木 康（慶應義塾大学）、中村 昌弘（レクサーリサーチ社）
Yasushi Kiyoki (Keio University) and Masahiro Nakamura (Lexer Research Co.)
データ間の意味的な関係の計量については、データ間の関係を静的かつ明示的に記述し、同一性、差異性を判定する方法が広く用いられてきました。しかし、これらの連想検索では、異なる表現形式であるが同一あるいは近い意味を持つデータを連想的に検索することはできません。慶應義塾大学清木研究室ではデータ間、あるいは、言葉とマルチメディアデータの意味的な関係、感性的な関係を文脈に応じて動的に計算するモデルとして、意味および感性を扱う意味の数学モデルの提案と実現を行なっており、本講演では、そのモデルをベースとしたクロスメディア型感性検索技術を紹介します。また、知的資産を利活用してゆく上では、散在するデジタル・アーカイブの検索、獲得、相互利用技術とともに、利用者での活用や構成を意思決定してゆくための利用者インターフェース環境も肝要です。レクサーでは自律的に挙動し、反応する機能性オブジェクト技術や空間的編集技術を基礎に、情報表現機構や直観的、体験認知的ユーザ・インターフェースの研究を行なっています。デジタル・アーカイブ利活用において教育分野に適用するためのユーザインターフェースとしての可能性や、ネットワーク・コラボレーションの構築例などを紹介します。

Associative search methods have been proposed for realizing efficient information retrieval in database and knowledge base systems. In the previously proposed methods, relationships betweendata items are represented explicitly by static descriptions of connections. In those methods, from the view point of semantic representation of data items, the meanings of the data items are definitely fixed, and those data items are used as information with the fixed meanings. As a model for measuring semantic relationships dynamically between data items and multimedia data according to contexts, the "Mathematical Model of Meaning" has been designed and developed in Kiyoki Laboratory, Keio University. In this presentation. we introduce semantic associative search technology for realizing cross-media data "Kansei" retrieval by applying this model. Furthermore, regarding to utilizing the intellectual property, the human interface technology for decision making is important in addition to retrieval, acquisition and sharing technology. LEXER Research has been investigating the research of intuitive and experimental recognition interfaces based on functional object and spatial authoring technologies. This session will introduce the ability of these technologies in some educational field and show some examples for network collaboration as the future interface in the application of digital archive.
 
招待講演(I)
ハイブリッド型デジタルライブラリ・システム
A Hybrid Digital-Automated Library System
Vilas Wuwongse （アジア工科大学教授、タイ）
An attempt to develop a hybrid library system is presented. According to INSPIRAL, a hybrid library collects and provides access to both physical and electronic information resources. It retains an identifiable physical location, and provides a single online location as well, offering users a one-stop access point. The online environment also allows the hybrid library to provide access to remote resources, and to information about resources held elsewhere. The developed hybrid system combines together a library automation system adapted from Kowha, a digital library system modified from Greenstone, and an advanced user-interface based on TV2WEB which enables browsing of videos and other multimedia information resources. The combination leads to an efficient and user-friendly one-stop service of access to both physical and electronic information resources.
 
デジタルアーカイブ活用のためのメタデータフレームワーク
Metadata based framework for digital archive utilization
安達淳（国立情報学研究所）
Dr. Jun Adachi (National Institute of Informatics, Japan)
文化財を中心とするメタデータおよび画像等のコンテンツを教育のために活用するためのシステムを提案します。内容、記述レベル、品質などが多様で精粗のあるメタデータを教育目的に活用するには、メタデータの進化、アノテーションとの統合、オーサリング、知識発見とマイニング等の機能要素を含む活用システムが必要です。我々の研究グループの持つ検索や利用者インターフェースの技術を紹介すると共に、提案する活用システムのねらいと実現方法について紹介します。

We propose a new system for further utilization of cultural digital contents and metadata in the educational environment. The metadata provided by various museums tend to be diversified in terms of content, description level and quality, and we have to consider the functions on the evolution of metadata, integration with annotation, authoring, knowledge discovery and data mining for educational utilization. We will talk on the aims of the system, implementation method, and the outline of the proposed system.
 
招待講演(II)
Knowledge Sifter：異種情報源に対する
エージェント指向・オントロジー駆動型サーチ
Knowledge Sifter: Agent-based Ontology-Driven Search
over Heterogeneous Data Sources
ラリー・カーシュバーグ
（米国ジョージ・メーソン大学教授、eビジネスセンター・ディレクタ）
Larry Kerschberg
(Professor and Co-Director, E-Center for E-Business, George Mason University)
Knowledge Sifter is a scaleable agent-based system that supports access to heterogeneous information sources such as the Web, open-source repositories, XML-databases and the emerging Semantic Web. User query specification is supported by a user agent that accesses multiple ontologies using an integrated conceptual model expressed in the Web Ontology Language (OWL). A collection of cooperating agents supports interactive query specification and refinement; query decomposition; query processing; as well as result ranking and presentation. The Knowledge Sifter architecture is general and modular so that ontologies and information sources can be easily incorporated. A proof-of-concept implementation shows how Knowledge Sifter can search geo-spatial ontology services such as the USGS Geographic Names Information System (GNIS) and Princeton University’s WordNet as well as image databases including Lycos and TerraServer. Each Agent is implemented as a Web Service and the external sources are also accessed via Web Service Technology.
 
セマンティックWebによるコンテンツの統合活用
Contents information integration based on Semantic Web
津田 宏, 星合 忠（株式会社 富士通研究所）
Dr. Hiroshi Tsuda, Dr. Tadashi Hoshiai (Fujitsu Laboratories Ltd.)
セマンティックWebの枠組み(RDF,OWL)に基き、マルチメディア検索、感性検索など異種の検索結果を統合し、教育のためにさまざまな観点で活用できるためのシステムを提案します。本発表では、まず、我々のRDFメタデータ抽出、異種オントロジーのアラインメント(対応づけ)、統合検索の応用(KnowWho)技術を説明します。それらを発展させ組合せることで、本プロジェクトで提案する検索統合システムの実現方法を紹介します。

Based on the framework of the Semantic Web (RDF, OWL), we propose an integrated search and utilization system that integrates multi-media search engine, sensibility search engine, and so on. In this presentation, we first explain our existing related technologies such as RDF metadata extraction from texts, heterogeneous ontology alignment, and KnowWho system as an application of information integration. Second, we introduce a new integrated search architecture in this project.
